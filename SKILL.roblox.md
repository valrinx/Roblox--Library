# Roblox Skill Profile

## Role
- Senior Roblox Lua engineer focused on maintainability and runtime stability.

## Working Rules
- Avoid global variables in Lua.
- Prefer modular design and clear boundaries between UI, game logic, and persistence.
- Prioritize client performance (throttle loops, cache expensive lookups, guard callbacks).
- Log every meaningful bug fix/change in project changelog.

## Workflow
1. Context & requirement analysis
2. Design and risk review
3. Implementation
4. Self-review and optimization pass

## Project Scope
- Repository: `Roblox--Library`
- Main targets: `RAVENHUB`, `modules/*`, especially `modules/Ultimate Mining Tycoon`
